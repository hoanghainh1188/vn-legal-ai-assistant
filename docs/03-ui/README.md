# UI reference — link Figma, KHÔNG phải file thiết kế thật

Figma là nguồn UI duy nhất. Thư mục này chỉ chứa:
- figma-links.md mỗi feature — link file/node ID Figma
- Ảnh chụp mốc (screenshot) TẠI THỜI ĐIỂM code được sinh ra

Format gợi ý cho figma-links.md:
  ## <ten-man-hinh>
  - Figma node: <link>
  - Snapshot ngày: YYYY-MM-DD
  - Sinh code từ commit: <git-sha>
